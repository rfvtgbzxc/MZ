# Ver.1.0.2
[ダウンロード](https://raw.githubusercontent.com/nuun888/MZ/master/NUUN_AnimationFPSRate.js)

MVのアニメーションを再生するときのフレームレートを変更します。  
またアニメーションごとにフレームレートを変更できます。  
※4フレームで15FPS、2フレームで30FPS、1フレームで60FPS
